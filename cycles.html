LOOP оператор

Описание

В Oracle оператора LOOP используется тогда, когда вы не уверены сколько раз вы хотите
выполнить тело цикла или выполнить тело цикла, по крайней мере, один раз.

Синтаксис

LOOP
   {…statements…}
END LOOP;

Параметры и аргументы

statements - код выполняющийся при каждом прохождении через цикл.

FOR операторы

Описание

Цикл FOR LOOP в Oracle позволяет выполнить код повторно в течение фиксированного
количества раз.

Синтаксис

FOR loop_counter IN [REVERSE] lowest_number..highest_number
    LOOP
        {…statements…}
        
    END LOOP;

Параметры и аргументы

loop_counter — переменная счетчика цикла в цикле FOR LOOP.
REVERSE не является обязательным. Если REVERSE указано счетчик цикла будет считать
в обратном порядке.
lowest_number — начальное значение для loop_counter
highest_number — конечное значение для loop_counter.
statements — код выполняющийся при каждом прохождении через цикл.

CURSOR FOR LOOP

Описание

CURSOR FOR LOOP можно использовать тогда, когда вы хотите выгрузить и обработать
каждую запись курсора. CURSOR FOR LOOP завершится, когда все записи будут
извлечены в курсор.

Синтаксис

FOR record_index in cursor_name
    LOOP
        {…statements…}
    END LOOP;

Параметры и аргументы

record_index – индекс записи
cursor_name — имя курсора, из которого вы хотите извлечь записи.
statements — код выполняющийся при каждом прохождении через цикл.

WHILE LOOP

Описание

В Oracle цикл WHILE LOOP, будет выполняться до тех пор, пока условие истинно.

Синтаксис

WHILE condition
    LOOP
        {…statements…}
    END LOOP;

Параметры и аргументы

condition Проверяется condition в каждом прохождении цикла. Если condition принимает
значение TRUE, цикла выполняется. Если condition принимает значение FALSE
цикл прекращается.

EXIT ОПЕРАТОР

Описание

В Oracle, оператор EXIT наиболее часто используется для завершения цикла LOOP.

Синтаксис

EXIT [WHEN boolean_condition];

Параметры и аргументы

boolean_condition не является обязательным. Это условие выхода из цикла.
